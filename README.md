# עץ הזית — מערכת GRC ארגונית (Mockup)

פרוטוטייפ Frontend בלבד (ללא לוגיקה עסקית) של מערכת GRC ארגונית בעברית — רגולציה, ניהול סיכונים וציות.

## מסכים

אפליקציית עמוד אחד עם 7 מסכים (מעבר דרך תפריט הניווט):

1. **דף הבית** — תמונת מצב ארגונית, KPI, משימות לטיפול ופרויקטים בתהליך אישור
2. **פרויקטים** — שלבי אישור, ועדות והתקדמות
3. **סיכונים** — מפת סיכונים לפי חומרה והסתברות
4. **בקרות** — סטטוס מימוש וראיות
5. **ציות** — פערי ציות ועמידה לפי תחום
6. **דוחות** — דוחות אחרונים ומגמות לאורך זמן
7. **ניהול** — משתמשים, תפקידים ומסלולי אישור

## הרצה

לפתוח את `index.html` בדפדפן, או להרים שרת סטטי:

```bash
npx http-server -p 8123 .
```

## טכנולוגיות

- HTML + CSS + JavaScript וניל, ללא תלויות
- RTL מלא, פונט Heebo
- רספונסיבי — דסקטופ, טאבלט ומובייל
- ניווט תומך ב-hash‏ (`#projects`, `#risks` וכו')
